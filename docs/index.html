NYC Construction Permit Approval Duration

This project builds a regression modeling pipeline to predict how long it takes for a construction permit to be approved by the NYC Department of Buildings. Data is pulled live from the NYC Open Data API and analyzed across five modeling approaches, from baseline OLS through regularization and tree-based methods.

Business Context

Permit approval timelines are a core uncertainty in construction project planning. This analysis examines whether publicly available permit metadata — job type, work type, borough, and building classification — can reliably predict approval duration, and surfaces the limitations of the available features for that task.

Dataset

  • NYC Department of Buildings — Permit Issuance
  • NYC Open Data (Dataset ID: ipu4-2q9a)
  • 100,000 records pulled via Socrata API
  • Response variable: approval duration (days from filing to issuance)
  • 30 encoded predictors after feature engineering

Methods

  • API data acquisition via sodapy
  • Feature engineering: date parsing, approval duration computation, categorical encoding
  • Baseline OLS regression with coefficient interpretation
  • Regularization models: Ridge, Lasso, and Elastic Net with cross-validated alpha selection
  • Tree-based model: Random Forest with feature importance analysis

Final Results

  • Baseline OLS: RMSE = 51.95 | R² = 0.0054
  • Ridge: RMSE = 51.94 | R² = 0.0060
  • Lasso: RMSE = 51.90 | R² = 0.0075
  • Elastic Net: RMSE = 51.90 | R² = 0.0073
  • Random Forest: RMSE = 52.70 | R² = −0.0234
  • Selected model: Lasso (best R², 12 of 30 predictors retained)
